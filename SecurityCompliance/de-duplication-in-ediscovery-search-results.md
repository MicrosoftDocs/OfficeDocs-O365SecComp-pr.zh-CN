## <a name="how-duplicate-messages-are-identified"></a><span data-ttu-id="42176-110">如何标识重复邮件</span><span class="sxs-lookup"><span data-stu-id="42176-110">How duplicate messages are identified</span></span>

<span data-ttu-id="42176-111">Office 365 电子数据展示工具使用以下电子邮件属性的组合来确定邮件是否重复:</span><span class="sxs-lookup"><span data-stu-id="42176-111">Office 365 eDiscovery tools use a combination of the following email properties to determine whether a message is a duplicate:</span></span>
  
- <span data-ttu-id="42176-112">**InternetMessageId** -此属性指定电子邮件的 Internet 邮件标识符, 它是指特定邮件的特定版本的全局唯一标识符。</span><span class="sxs-lookup"><span data-stu-id="42176-112">**InternetMessageId** - This property specifies the Internet message identifier of an email message, which is a globally unique identifier that refers to a specific version of a specific message.</span></span> <span data-ttu-id="42176-113">此 ID 由发件人的电子邮件客户端程序或用于发送邮件的主机电子邮件系统生成。</span><span class="sxs-lookup"><span data-stu-id="42176-113">This ID is generated by the sender's email client program or host email system that sends the message.</span></span> <span data-ttu-id="42176-114">如果某人向多个收件人发送邮件, 则该邮件的每个实例的 Internet 邮件 ID 都是相同的。</span><span class="sxs-lookup"><span data-stu-id="42176-114">If a person sends a message to more than one recipient, the Internet message ID will be the same for each instance of the message.</span></span> <span data-ttu-id="42176-115">对原始邮件的后续修订将接收到不同的邮件标识符。</span><span class="sxs-lookup"><span data-stu-id="42176-115">Subsequent revisions to the original message will receive a different message identifier.</span></span> 
    
- <span data-ttu-id="42176-116">**ConversationTopic** -其属性指定邮件的会话线程的主题。</span><span class="sxs-lookup"><span data-stu-id="42176-116">**ConversationTopic** - his property specifies the subject of the conversation thread of a message.</span></span> <span data-ttu-id="42176-117">**ConversationTopic**属性的值是描述对话的总体主题的字符串。</span><span class="sxs-lookup"><span data-stu-id="42176-117">The value of the **ConversationTopic** property is the string that describes the overall topic of the conversation.</span></span> <span data-ttu-id="42176-118">保留项包括初始邮件和在答复初始邮件时发送的所有邮件。</span><span class="sxs-lookup"><span data-stu-id="42176-118">A conservation consists of an initial message and all messages sent in reply to the initial message.</span></span> <span data-ttu-id="42176-119">同一对话中的邮件具有与**ConversationTopic**属性相同的值。</span><span class="sxs-lookup"><span data-stu-id="42176-119">Messages within the same conversation have the same value for the **ConversationTopic** property.</span></span> <span data-ttu-id="42176-120">此属性的值通常是生成对话的初始邮件中的主题行。</span><span class="sxs-lookup"><span data-stu-id="42176-120">The value of this property is typically the Subject line from the initial message that spawned the conversation.</span></span> 
    
- <span data-ttu-id="42176-121">**BodyTagInfo** -这是一个内部 Exchange 存储属性。</span><span class="sxs-lookup"><span data-stu-id="42176-121">**BodyTagInfo** - This is an internal Exchange store property.</span></span> <span data-ttu-id="42176-122">此属性的值是通过检查邮件正文中的各种属性计算得出的。</span><span class="sxs-lookup"><span data-stu-id="42176-122">The value of this property is calculated by checking various attributes in the body of the message.</span></span> <span data-ttu-id="42176-123">此属性用于标识邮件正文中的差异。</span><span class="sxs-lookup"><span data-stu-id="42176-123">This property is used to identify differences in the body of messages.</span></span> 
    
<span data-ttu-id="42176-124">在电子数据展示导出过程中, 将对匹配搜索条件的每封邮件比较这三个属性。</span><span class="sxs-lookup"><span data-stu-id="42176-124">During the eDiscovery export process, these three properties are compared for every message that matches the search criteria.</span></span> <span data-ttu-id="42176-125">如果两个 (或更多) 邮件的这些属性相同, 则确定这些邮件是重复的, 并且结果是, 如果启用重复数据消除功能, 则将只导出邮件的一个副本。</span><span class="sxs-lookup"><span data-stu-id="42176-125">If these properties are identical for two (or more) messages, those messages are determined to be duplicates and the result is that only one copy of the message will be exported if de-duplication is enabled.</span></span> <span data-ttu-id="42176-126">导出的邮件称为 "源项目"。</span><span class="sxs-lookup"><span data-stu-id="42176-126">The message that is exported is known as the "source item".</span></span> <span data-ttu-id="42176-127">有关重复邮件的信息包含在导出的搜索结果附带的**结果 .csv**和**清单 .xml**报告中。</span><span class="sxs-lookup"><span data-stu-id="42176-127">Information about duplicate messages is included in the **Results.csv** and **Manifest.xml** reports that are included with the exported search results.</span></span> <span data-ttu-id="42176-128">在**结果 .csv**文件中, 通过在 "**复制到项目**" 列中包含一个值来标识重复的邮件。</span><span class="sxs-lookup"><span data-stu-id="42176-128">In the **Results.csv** file, a duplicate message is identified by having a value in the **Duplicate to Item** column.</span></span> <span data-ttu-id="42176-129">此列中的值与导出的邮件的 "**项目标识**" 列中的值相匹配。</span><span class="sxs-lookup"><span data-stu-id="42176-129">The value in this column matches the value in the **Item Identity** column for the message that was exported.</span></span> 
  
<span data-ttu-id="42176-130">下图显示了在搜索结果中导出的**结果 .csv**和**清单 .xml**报告中显示重复邮件的方式。</span><span class="sxs-lookup"><span data-stu-id="42176-130">The following graphics show how duplicate messages are displayed in the **Results.csv** and **Manifest.xml** reports that are exported with the search results.</span></span> <span data-ttu-id="42176-131">这些报告不包括前面所述的电子邮件属性, 这些属性在重复数据消除算法中使用。</span><span class="sxs-lookup"><span data-stu-id="42176-131">These reports don't include the email properties previously described, which are used in the de-duplication algorithm.</span></span> <span data-ttu-id="42176-132">相反, 报告包括分配给 Exchange 存储中的项目的**项目标识**属性。</span><span class="sxs-lookup"><span data-stu-id="42176-132">Instead, the reports include the **Item Identity** property that is assigned to items by the Exchange store.</span></span>
