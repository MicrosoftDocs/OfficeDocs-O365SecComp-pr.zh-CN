# <a name="get-started-with-the-microsoft-service-trust-portal"></a>Microsoft 服务信任门户入门

在 Microsoft 服务信任门户提供了多种内容、 工具和有关 Microsoft 安全、 隐私和合规性做法其他资源。
  
## <a name="accessing-the-service-trust-portal"></a>访问服务信任门户

在服务信任门户包含有关 Microsoft 的控制和保护我们云服务和客户数据其中的过程实现的详细信息。若要访问某些服务信任门户上的资源，必须使用 Microsoft 云服务帐户 （Azure Active Directory 组织帐户或 Microsoft 帐户），以经过身份验证的用户登录并查看并接受 Microsoft保密协议合规性材料。
  
在**现有客户可以访问服务信任门户**[https://aka.ms/STP](https://aka.ms/STP)具有以下 online 订阅 （试用版或付费） 之一： 
  
- Office 365

- Dynamics 365

- Azure
    
 > [!NOTE]
 > 与组织关联的 azure Active Directory 帐户有权访问的文档和功能像合规性管理器中的全部。创建个人使用 Microsoft 帐户具有有限访问服务信任门户内容。 
  
**新的客户和评估 Microsoft online 服务的客户**
  
若要创建新帐户，或创建试用帐户，使用以下某种形式注册 （还用于试用帐户） 获取 STP 访问。
  
- 注册新的[Office 365 企业版试用帐户](https://go.microsoft.com/fwlink/p/?LinkID=507653)或新的[Office 365 企业版试用帐户](https://go.microsoft.com/fwlink/p/?LinkID=698279)

- 注册新的[Dynamics 365 试用帐户](https://go.microsoft.com/fwlink/?LinkId=252780)

- 注册新的[Azure 试用帐户](https://go.microsoft.com/fwlink/?LinkId=722737)。 
    
当您注册免费试用版，或订阅时，您必须启用 Azure Active Directory 以支持您对 STP 访问权限。
  
## <a name="using-the-service-trust-portal"></a>使用服务信任门户

服务信任门户功能和内容进行访问从主菜单。
  
![服务信任门户菜单](media/STPMenus1.png)

以下各节介绍主菜单中的每个项目。
  
### <a name="service-trust-portal"></a>服务信任门户

**服务信任门户**链接显示的主页。它提供了一种以返回到主页页的快捷方式。

### <a name="compliance"></a>合规性

提供了下列法规遵从性工具和资源：

- 显示**审核报告**-Microsoft 云服务上的独立审核和评估报告的列表。这些报告提供信息 Microsoft 云服务合规性数据保护标准和法规要求，如：
  
    - 国际标准化 (ISO) 组织
    - 服务组织控制 (SOC)
    - 国家标准和技术 (NIST)
    - 联邦风险和授权管理程序 (FedRAMP)
    - 常规数据保护法规 (GDPR)

  在服务信任门户主页上还显示有关审核报告和**审核报表**页面的链接的信息。 
  
- **合规性管理器**-合规性管理器是一个基于工作流的风险评估工具，可帮助您跟踪、 分配，并确认贵组织的法规遵从性活动 Dynamics 365 与 Microsoft 云服务，例如 Office 365 和Azure。使用合规性管理器管理的云中共享的责任模型中的法规遵从性。有关详细信息，请参阅[使用合规性管理器，以帮助满足数据保护和管理法规要求时使用 Microsoft 云服务](meet-data-protection-and-regulatory-reqs-using-microsoft-cloud.md)。

### <a name="industries--regions"></a>行业 & 区域

提供有关 Microsoft 云服务的特定于行业和区域合规性信息。

- **行业**-提供特定于行业登陆页面的信息以及有关 Microsoft 云服务如何帮助组织符合标准和法规对这些行业：

    - 教育版
    - 金融服务
    - 政府
    - 运行状况
    - 制造
    - 零售

- **区域**-提供有关 Microsoft 云服务承担遵守各种法律各种国家/地区的法律见解。澳大利亚、 捷克共和国、 德国、 波兰、 罗马尼亚、 西班牙和英国，包括特定国家/地区。
  
### <a name="documents--resources"></a>文档 & 资源

提供了丰富的安全实施和设计信息目的是为了使您能够通过了解 Microsoft 云服务如何保护数据安全满足法规遵从性目标更加方便。若要查看内容，请选择**文档 & 资源**页上的选项之一。

- 白皮书、 常见问题解答和 How 到文章
- 合规性指南
- 笔测试和安全评估
- 安全分数
- Azure 安全性和合规性蓝图
- 审核的控件
  
### <a name="trust-center"></a>信任中心

对[Microsoft 信任中心](https://www.microsoft.com/trustcenter)，它提供有关安全、 合规性、 和 Microsoft 云中的隐私的详细信息的链接。这包括有关功能的信息可用于解决特定需求 GDPR，文档有用到 GDPR 责任和技术和组织度量您了解 Microsoft 云服务中Microsoft 具有所需支持 GDPR。
  
### <a name="my-library"></a>我库

此新功能允许您保存 （或*pin*） 的文档，以便您可以快速访问其在我库页上。您还可以设置通知，以便 Microsoft 向您发送一封电子邮件更新我库中的文档时。有关详细信息，请参阅本文中的[我的库](#my-library-1)部分。

### <a name="admin"></a>管理员

仅对全局管理员帐户是可用的管理功能。此选项是可见的仅当您以全局管理员身份登录。
  
 **设置**页允许您分配到合规性管理器中的基于角色的访问。有关详细信息，请参阅[使用合规性管理器，以帮助满足数据保护和管理法规要求时使用 Microsoft 云服务](meet-data-protection-and-regulatory-reqs-using-microsoft-cloud.md#permissions-and-role-based-access-control)中的"权限和基于角色的访问控制"部分。

## <a name="my-library"></a>我库

使用我库功能添加到我的库页面服务信任门户中的文档和资源。使您能够访问与您在一个位置相关的文档。 将文档添加到您我库中，单击右侧的文档 **...** 菜单，然后选择**添加到库**。可以通过单击一个或多个文档旁边的复选框，然后单击**保存到库**页的顶部到您我库中添加多个文档。

此外，通知功能允许您配置您我的库，以便向您发送一封电子邮件，只要 Microsoft 更新已添加到您我的库的文档。若要设置通知，转到您我的库，然后单击**通知设置**。您可以选择通知的频率，并向其发送通知您组织中指定的电子邮件地址。电子邮件通知包括已更新的文档的链接和更新的简要说明。

另请注意，我们识别我库中的已更新最近 30 天，无论您打开通知中的文档。此外会将更新的简要说明显示工具提示中。 

## <a name="starter-packs"></a>初学者包

初学者包是文档的一组 Microsoft curated 有关针对特定行业 Microsoft 云服务。目前，在服务信任门户提供以下三个 starter 包的金融服务组织。这些 starter 包帮助评估评估安全、 合规性、 和 Microsoft 云中的隐私和提供的指导可帮助高度管控金融服务行业中实现 Microsoft 云服务的组织。

- **评估 Starter 包**-用于 Microsoft 云金融服务组织的最早评估。

- **评估 Starter Pack** -后评估，使用清单和其他指南此 starter 包中以帮助组织评估与安全、 合规性、 和隐私相关的风险。

- **审核 Starter Pack** -用户有关使用审核控件和其他工具以帮助的 Microsoft 云服务的方式，可帮助您实现指南此 starter 包减少组织的风险的。

若要访问这些 starter 包，请转到**服务信任门户 > 行业 & 区域 > 行业解决方案 > 金融服务**。可以打开或下载文档中的初学者包，将其保存到您我的库。
