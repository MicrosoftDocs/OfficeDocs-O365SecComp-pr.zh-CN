<span data-ttu-id="0e305-p111">当某人在 Exchange Online 中发送与加密邮件流规则匹配的电子邮件时, 邮件会在发送之前进行加密。所有使用 Outlook 客户端的 Office 365 最终用户读取邮件接收对加密和受权限保护的邮件的本机、第一类阅读体验, 即使它们与发件人不在同一组织中也是如此。支持的 outlook 客户端包括 outlook 桌面、outlook Mac、iOS 和 Android 上的 outlook 移动以及 web 上的 outlook (以前称为 outlook web App)。</span><span class="sxs-lookup"><span data-stu-id="0e305-p111">When someone sends an email message in Exchange Online that matches an encryption mail flow rule, the message is encrypted before it's sent.
